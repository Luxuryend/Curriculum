# 各种寻址方式的原理；
| 寻址方式         | 有效地址(EA)计算公式                          |
|------------------|---------------------------------------------|
| 立即寻址         | 不涉及内存地址，操作数直接在指令中           |
| 直接寻址         | EA = 指令中的地址字段（A）                   |
| 寄存器寻址       | 操作数在寄存器中，无需地址计算               |
| 寄存器间接寻址   | EA = (寄存器内容)                           |
| 基址寻址         | EA = 基址寄存器 + 位移量（A）                |
| 变址寻址         | EA = 变址寄存器 + 位移量（A）                |
| 基址+变址寻址    | EA = 基址寄存器 + 变址寄存器 + 位移量        |
| 相对寻址         | EA = 当前PC值 + 相对偏移量                   |
| 间接寻址         | EA = (地址字段A)                            |

# DMA 的三种传送方式特点；
CPU暂停方式：控制简单，资源没有充分利用  
CPU周期窃取方式：提高CPU利用率，拖慢CPU执行速度  
DMA与CPU交替访问：并行工作，吞吐率高，结构复杂

# 中断的概念，中断的执行过程。

开中断 -> 执行中断服务程序 ->  
关中断 -> 恢复现场恢复断点 ->  
开中断 -> 返回断点
