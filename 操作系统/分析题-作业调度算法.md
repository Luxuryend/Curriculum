# 题目
假定在单CPU条件下,有A,B,C,D四个作业依次到达(后面的作业依次比前一作业迟到一个时间单位)。四个作业分别需要运行11,6,2和1个时间单位
| 作业 | 到达时间 | 运行时间 | 开始时间 | 完成时间 | 周转时间 | 带权周转时间 |
| :---: | :---: | :---: | :---: | :---: | :---: | :---: |
| A    | 0      | 11     |        |        |        |            |
| B    | 1      | 6      |        |        |        |            |
| C    | 2      | 2      |        |        |        |            |
| D    | 3      | 1      |        |        |        |            |

# 解答
开始时间=上一作业的完成时间  
完成时间=运行时间+开始时间  
周转时间=完成时间-到达时间  
带权周转时间=周转时间/运行时间

按照到达次序来算，也就是A-B-C-D

## 高响应比优先 HRRN

$$
响应比RR = 1 + \frac{等待时间(上一个作业完成时间-到达时间)}{运行时间}
$$
