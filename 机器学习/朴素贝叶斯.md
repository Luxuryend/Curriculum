# 贝叶斯分类预测计算过程

## 题目
使用贝叶斯分类方法预测一个客户（性别=女，婚姻状态=已婚，是否有房=无房），是否会购买此保险，写出详细的计算过程。

| 记录 | 性别 | 婚姻状态 | 是否有房 | 购买保险 |
| :---: | :---: | :---: | :---: | :---: |
| 1 | 女 | 未婚 | 无 | 否 |
| 2 | 女 | 未婚 | 有 | 否 |
| 3 | 女 | 已婚 | 有 | 否 |
| 4 | 女 | 已婚 | 有 | 否 |
| 5 | 女 | 未婚 | 有 | 是 |
| 6 | 男 | 未婚 | 有 | 是 |
| 7 | 男 | 未婚 | 有 | 否 |
| 8 | 男 | 已婚 | 有 | 否 |
| 9 | 男 | 未婚 | 有 | 是 |
| 10 | 男 | 未婚 | 有 | 是 |

---

## 解答
设会买保险为 $C_1$，不会买保险为 $C_2$。

### 1. 计算类先验概率
- $P(C_1) = 4/10 = 2/5$
- $P(C_2) = 6/10 = 3/5$

### 2. 待分类数据
待分类数据为 $X = (\text{性别=女}，\text{婚姻状态=已婚}，\text{是否有房=无})$

### 3. 计算各项条件概率
**对于 $C_1$：**
- $P(\text{性别=女} | C_1) = 1/4$
- $P(\text{婚姻状态=已婚} | C_1) = 0/4 = 0$
- $P(\text{是否有房=无} | C_1) = 0/4 = 0$

**对于 $C_2$：**
- $P(\text{性别=女} | C_2) = 4/6 = 2/3$
- $P(\text{婚姻状态=已婚} | C_2) = 3/6 = 1/2$
- $P(\text{是否有房=无} | C_2) = 1/6$

### 4. 计算条件概率 $P(X|C_i)$
- $P(X | C_1) = 1/4 \times 0 \times 0 = 0$
- $P(X | C_2) = 2/3 \times 1/2 \times 1/6 = 1/18$

### 5. 最终结果对比
- $P(X | C_1)P(C_1) = 0 \times 2/5 = 0$
- $P(X | C_2)P(C_2) = 1/18 \times 3/5 = 1/30$

### 6. 结论
因为 $0 < 1/30$，所以该用户预测结果为：**不会购买保险**。
