## 第一轮

### 天气  
晴 vs 阴+雨  
晴(4)：是(1)、否(3) → Gini = 1 - (1/4)² - (3/4)² = 3/8  
阴+雨(6)：是(4)、否(2) → Gini = 1 - (4/6)² - (2/6)² = 4/9  
加权Gini = 4/10·3/8 + 6/10·4/9 = **5/12 ≈ 0.416**  

阴 vs 晴+雨  
阴(3)：是(3)、否(0) → Gini = 0  
晴+雨(7)：是(2)、否(5) → Gini = 1 - (2/7)² - (5/7)² = 20/49  
加权Gini = 3/10·0 + 7/10·20/49 = **2/7 ≈ 0.285**  

雨 vs 晴+阴  
雨(3)：是(1)、否(2) → Gini = 4/9  
晴+阴(7)：是(4)、否(3) → Gini = 24/49  
加权Gini = 3/10·4/9 + 7/10·24/49 = **10/21 ≈ 0.476**  

#### 湿度
高湿度(5)：是(1,3) → 是(1次)、否(4次) → Gini = 1 - (1/5)² - (4/5)² = **8/25 = 0.32**  
正常湿度(5)：是(5,7,9,10) → 是(4次)、否(1次) → Gini = 1 - (4/5)² - (1/5)² = **8/25 = 0.32**  
加权Gini = (5/10×8/25)+(5/10×8/25) = **8/25 ≈ 0.32**  

 
#### 有风
有风(3)：是(7) → 是(1次)、否(2次) → Gini = 1 - (1/3)² - (2/3)² = **4/9 ≈ 0.444**  
无风(7)：是(3,5,9,10) → 是(4次)、否(3次) → Gini = 1 - (4/7)² - (3/7)² = **24/49 ≈ 0.489**  
加权Gini = (3/10×4/9)+(7/10×24/49) ≈ **0.476**
